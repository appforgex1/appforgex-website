# Contributing to Appforgex Digital Hub

Thank you for your interest in contributing to the Appforgex Digital Hub! As a proprietary project, contributions are typically handled by our internal team or authorized partners.

## How to Contribute

1. **Reporting Bugs**: If you find a bug, please open an issue describing the problem and steps to reproduce it.
2. **Feature Requests**: We welcome suggestions for new features. Please open an issue to start a discussion.
3. **Internal Proposals**: If you are part of the internal development team, please follow our standard branching and pull request workflow.

## Development Standards

- **TypeScript**: Ensure all new code is strictly typed.
- **Components**: Use our existing Shadcn-based component library.
- **Aesthetics**: Maintain the premium, dark-themed visual design.
- **Testing**: Run `npm run build` before submitting any changes.

## Security

Please report any security vulnerabilities directly to hello@appforgex.com. Do not open public issues for security concerns.

Thank you for helping us build the future of digital solutions!
