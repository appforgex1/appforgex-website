# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.1.0] - 2026-02-11

### Added
- Integrated Google Forms for contact requests.
- Implemented bot protection via Honeypot field.
- Added client-side rate limiting for form submissions.
- Professional project documentation: `LICENSE`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`.
- Increased message character limit to 10,000 with real-time counter UI.

### Changed
- Refined contact form UI with better field labels and interactive states.
- Enhanced form validation using Zod.

## [1.0.0] - 2026-02-10

### Added
- Initial release of Appforgex Digital Hub.
- Modern landing page with premium animations.
- Services, Industries, and Portfolio sections.
- Responsive design and dark mode.
